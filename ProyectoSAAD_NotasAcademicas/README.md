# Proyecto SAAD: Sistema de Notas Académicas

Este proyecto implementa el modelado y análisis de un **Sistema de Alerta y Análisis de Datos (SAAD)** enfocado en la gestión de notas académicas.

El objetivo es aplicar los conceptos de análisis estructurado, diseño orientado a objetos y análisis de datos para crear los planos de ingeniería de un sistema funcional.

## Fases del Proyecto

1.  **Fase I - Requerimientos y Flujo:** Creación del DFD y Diccionario de Datos.
2.  **Fase II - Diseño Lógico:** Creación de Pseudocódigo/Diagrama de Flujo.
3.  **Fase III - Modelado POO:** Creación del Diagrama de Clases UML.
4.  **Fase IV - Análisis de Datos:** Implementación de un análisis con Python y despliegue en Streamlit.
